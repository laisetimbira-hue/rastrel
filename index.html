Configurações
Firebase, operador e leitor RFID
Firebase

Acesse console.firebase.google.com → seu projeto → ⚙️ Configurações → SDK e copie os dados abaixo.

Operador
Leitor RFID — modo HID
1
Configure o leitor no modo HID Keyboard (veja o manual do seu leitor)
2
Pareie via Bluetooth nas configurações do celular Android
3
Na aba Vincular, toque no campo EPC azul para focar
4
Aponte o leitor para o bag — o código aparece automaticamente ✅
💡 Dica
Sempre toque no campo EPC antes de usar o leitor para garantir que o foco está correto.
Zona de perigo

Apaga permanentemente todos os bags, lotes, movimentações e pedidos de carga salvos no banco de dados. Use isso só para limpar dados de teste antes de começar a operação real.
